IoT in Sämsjön

As part of a development project at Herrljunga Municipality in 2024–25, we built an API that connects to an IoT device located in a buoy in Lake Sämsjön. The system measures water temperature, stores the data in a SQL database, and presents it on the municipality’s official website.

At the moment, only a single line with the temperature is displayed on the site, and it's somewhat easy to miss — but if you scroll down, you'll find it just above the heading for the Sandsken swimming area.

QMS in Microsoft Power Platform

In the spring of 2025, I was the project manager for developing a prototype of a Quality Management System for the Municipal Management Team in Herrljunga.

I led the work from start to finish organizing workshops to gather input from the management team, designing the prototype in Microsoft Power Apps, integrating it with SharePoint and Power Automate, and presenting the final result. I ensured progress, solved problems, and became the go-to person throughout the process.

Quizdan - Test Your Knowledge of Sweden’s Democracy!

"Quizdan" is a game build with collecting information and pictures through Sweden's goverment's API and making it a fun game! We were a group of four java coders and four UX-designers who created the game from scratch for a school project. And winning for the most creative and best concept of a game the year of 2024!
