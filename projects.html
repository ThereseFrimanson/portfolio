IoT in Sämsjön

As part of a development project at Herrljunga Municipality in 2024–25, we built an API that connects to an IoT device located in a buoy in Lake Sämsjön. The system measures water temperature, stores the data in a SQL database, and presents it on the municipality’s official website.

At the moment, only a single line with the temperature is displayed on the site, and it's somewhat easy to miss — but if you scroll down, you'll find it just above the heading for the Sandsken swimming area.
